# ver 4 추가 수정 사항

### 1. 레이아웃 너비 복원

- 패딩 조정으로 더 넓은 작업 공간 확보

### 2. Header 로고 글씨 크기 확대

- 아이콘 크기: text-xl → **text-2xl**

- 서브 텍스트: text-lg → **text-xl**
- 드롭존 제목: text-xl → **text-2xl**
- 피처 아이템 제목: text-base → **text-lg**
- 피처 아이템 설명: text-sm → **text-base**

- 모든 `dark:` 접두사 클래스 제거
- 일관된 화이트 테마 적용
