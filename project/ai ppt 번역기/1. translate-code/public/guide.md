# Safety Translation Guide (한-영 번역 통합 가이드)

이 문서는 한–영 번역 시 일관되고 전문적인 결과를 위해 사용하는 통합 가이드입니다. 
보고서/발표/캠페인 등 문서 유형별 스타일, 형식 규칙, 용어집, 워크플로우, 체크리스트를 포함합니다.

---

## 1. 기본 원칙 (General Rules)

- **문체**: 포멀·객관적 보고서 톤 유지 (비즈니스 리포트/분석 보고서 수준)
- **용어 일관성**: 한 용어 = 하나의 영어 표현 (문서 전반 동일 표현 유지)
- **의역 우선**: 불필요한 직역 금지, 의미 중심 자연스러운 영문화
- **강조 유지**: 굵게(Bold), 밑줄(Underline) 등 강조 표시는 원문대로 반영
- **수치/단위 유지**: 숫자, 단위(%, 인원, pp, ▲/▼ 등) 그대로 유지
  - %: 퍼센트 (percent)
  - pp: 퍼센트포인트 (percentage point)
- **문장 재배치**: 영어 문법상 자연스럽게 어순 조정
- **문장 분할**: 긴 문장은 영어식으로 2문장 이상으로 분리 가능
- **고유명사**: 회사명·현장명은 번역하지 않음

---

## 2. 문서 유형별 문체 (Document Type Style)

### 보고서 / 분석자료
- Formal business English
- 완전한 문장 구조
- 논리 연결어 적절 사용

### 발표자료 (PPT)
- 간결·핵심 키워드 중심
- 불필요 수식어 축소
- 줄글 지양
- **주의**: 한국어→영어 변환 시 텍스트 길이 증가로 레이아웃 붕괴 위험
  - 동의어 활용 및 축약으로 길이 최소화

### 캠페인 / 포스터
- 행동 유도형 표현 (예: Stay alert, stay safe!)

---

## 3. 분기 표기 (Quarter Notation)

- 1분기 → **1Q**
- 2분기 → **2Q**
- 3분기 → **3Q**
- 4분기 → **4Q**

## 5. 형식 규칙 (Formatting Rules)

### 표(Table) Column 명칭
- 표준 컬럼: `Question` / `Content` / `Category` / `Q1` / `Q2` / `Remarks`

### 제목 스타일
- 형식: `번호 + 마침표 + 공백 + 제목`
- 예시: `02. Key Survey Results`

### 불릿포인트 규칙
- **원본 파일(한글) 유지 원칙**: 원본 한글 파일의 불릿포인트 형식을 그대로 보존
- **영문 번역 시 적용 규칙**:
  - 하이픈(-) 사용
  - 첫 단어 소문자 시작
  - 끝에 마침표 없음
  - 예시: `- awareness improved compared to 1Q`

### 기호/단위
- ▲ / ▼ / % / pp 수치와 함께 유지

---

## 6. 번역 절차 (Translation Flow)

1. **문서 유형 파악**: 보고서/분석형/캠페인형 식별
2. **사전 우선**: 내부 고정 용어(안전, 점검, 교육 등) 우선 적용
3. **강조 반영**: Bold/Underline 등 시각적 강조 그대로 유지
4. **수치·단위 유지**: 원문 숫자/단위 그대로
5. **고유명사 비번역**: 회사/현장/캠페인명 그대로 사용
6. **문장 최적화**: 긴 문장은 분리. PPT는 길이 축소 및 핵심화

## 8. 분야별 지침 (Domain-Specific Rules)

### Safety 분야
- OSHA 및 산업안전보건법 기준 용어 사용

### Survey 분석
- 문항 번호 `Qx-x` 원형 유지
- 문장 자체는 자연스럽게 의역

### Campaign 명
- 고유명사 취급
- 영문 우선 표기: Point Me Out
- 필요 시 한글 병기: Point Me Out(포인트 미 아웃)
- "캠페인" 단어는 문맥상 필요 시만 추가

### Statistics
- ▲, ▼, %, pp 등 기호와 수치 함께 유지

## 10. 표준 테이블 스키마 (Table Schema Standard)

### Column 이름 (영문 고정)
`Question` / `Content` / `Category` / `Q1` / `Q2` / `Remarks`

### 예시 (행)
| Question | Content | Category | Q1 | Q2 | Remarks |
|----------|---------|----------|----|----|---------|
| Q1-1 | compliance with safety rules | Behavior | 78% | 81% | ▲ 3pp vs 1Q |

- 불릿/문장 규칙 준수: 첫 단어 소문자, 문장 끝 마침표 없음

## 11. 제목/불릿 예시 (Title & Bullet Examples)

### 제목
```
02. Key Survey Results
```

### PPT 불릿
```
- awareness improved compared to 1Q
- near-miss reporting increased by 12%
- vfl activities expanded at the Remicon Division
```

### 보고서 문장 (풀센텐스)
```
Safety compliance improved by 3pp compared to 1Q. The upward trend was consistent across all divisions.
```

## 12. 통계 표기 규칙 (Statistics Notation)

### 증감 표기
- **상승**: ▲ (숫자와 함께)
  - 예: `▲ 2pp`, `▲ 1.3pp`
- **하락**: ▼ (숫자와 함께)
  - 예: `▼ 2pp`, `▼ 1.3pp`

### 비교 기준
- vs 1Q, vs 2024, YoY, QoQ 등 필요 시 사용

### 단위 유지
- %, pp, 명 등 원문 단위 유지

## 13. 품질 체크리스트 (Quality Checklist)

- [ ] 문서 유형에 맞는 톤/길이인가? (보고서 vs PPT vs 캠페인)
- [ ] 고정 용어 표준 표기를 일관 적용했는가?
- [ ] Bold/Underline 등 강조 보존되었는가?
- [ ] 숫자/단위/기호(%, pp, ▲/▼) 그대로 유지했는가?
- [ ] 회사명/현장명/캠페인명 비번역했는가?
- [ ] 긴 문장은 영어식으로 분리/축약했는가?
- [ ] 표(Column 명)와 제목 규칙을 준수했는가?
- [ ] 문항 번호(Qx-x) 원형 유지했는가?

## 15. 예시 변환 (Before → After)

### PPT 변환 예시

**원문 (PPT)**
```
1분기 대비 안전수칙 준수율이 3%p 상승했습니다. 각 부문에서 개선세가 확인되었습니다.
```

**번역 (PPT)**
```
- safety compliance increased by 3pp vs 1Q
- improvement observed across all divisions
```

### 보고서 변환 예시

**원문 (보고서)**
```
안전문화 인식은 전반적으로 개선되었으나, 아차사고 보고는 특정 현장에서 정체되었습니다.
```

**번역 (보고서)**
```
Safety culture awareness improved overall. However, near-miss reporting plateaued at specific sites.
```

## 16. 예외/특기 사항 (Notes & Exceptions)

- **법률/안전 용어**: 내부 첨부 단어사전을 반드시 우선 적용
- **캠페인명**: 고유명사로 간주. 필요 시 영문(한글 병기) 가능
  - 예: `Point Me Out(포인트 미 아웃)`
- **수치 표기**: 길어질 경우 표/불릿 우선. 본문에서는 핵심만 요약

## 요약 (Summary)

| 항목 | 원칙 |
|------|------|
| **톤** | 포멀, 객관, 비즈니스 |
| **길이** | 보고서는 완전 문장, PPT는 핵심 키워드 |
| **형식** | 표 Column/제목/불릿 규칙 엄수 |
| **용어** | 표준 표기 일관 적용, 고유명사 비번역 |
